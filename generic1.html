Data Cleaning

Data cleaning is the process of deleting or correcting erroneous, corrupted, or badly structured data, as well as removing duplication within a dataset (IBM,n.d.) These problems are conceivable whenever data is combined or exported into a new format.

Detecting labelling or duplication problems can be challenging when dealing with big amounts of data. According to (Alteryx, n.d.) data cleaning strategy can assist in reducing or eliminating these errors, ensuring that you have the high-quality data you require to make data-driven decisions.

Through out this module, we employed data cleaning techniques to clean data obtained from different sources during our projects. In Unit 11, our database build project, used web scraping techniques to scrap data from Amazon and eBay websites. Since the data was gathered through scraping websites, it was captured in an unclean form. Data cleaning using python libraries like Pandas was used to clean the data from errors, mistakes, and redundancies.
